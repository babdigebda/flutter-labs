1.Тема: 
Создание приложения заметок
2.Функции:
Просмотр списка заметок: Отображение всех сохраненных заметок в виде списка на главном экране.
Создание заметки: Возможность добавить новую заметку, указав заголовок и основной текст.
Редактирование заметки: Возможность изменить заголовок или текст существующей заметки.
Удаление заметки: Удаление ненужной заметки из списка.
Просмотр курсов валют: Интеграция с публичным API для получения курса валют. Fixer — это простой и легковесный API для получения текущих и исторических курсов валют (форекс).

4.Сценарии использования:
Сценарий 1: Создание новой заметки
1.  Пользователь открывает приложение и попадает на главный экран со списком заметок.
2.  Он нажимает на плавающую кнопку «+» в правом нижнем углу.
3.  Приложение выполняет переход на экран создания/редактирования заметки. Поля ввода пусты.
4.  Пользователь вводит заголовок и содержание заметки.
5.  Для сохранения он нажимает на иконку «Сохранить» в верхнем меню.
6.  Приложение возвращается на главный экран, и новая заметка появляется в списке.

Сценарий 2: Редактирование существующей заметки
1.  На главном экране пользователь нажимает на карточку заметки, которую хочет изменить.
2.  Открывается экран редактирования, где поля уже заполнены текущими данными заметки.
3.  Пользователь вносит изменения в текст.
4.  Нажимает иконку «Сохранить».
5.  Приложение возвращается на главный экран, а текст заметки в списке обновляется.

Сценарий 3: Удаление заметки
1.  На главном экране пользователь находит заметку, которую хочет удалить.
2.  Он нажимает на иконку «Корзина», расположенную справа на карточке заметки.
3.  Появляется диалоговое окно с подтверждением: «Удалить заметку?».
4.  Пользователь подтверждает удаление.
5.  Заметка исчезает из списка на главном экране.

Сценарий 4: Получение кура валют
1.  На главном экране пользователь нажимает на иконку «$» в верхнем меню.
2.  Приложение переходит на экран курса валют.
3.  Пользователь нажимает кнопку «$».
4.  Приложение отправляет запрос к публичному API Fixer, и на несколько секунд появляется индикатор загрузки.
5.  В области для текста отображается полученный курс валют.
